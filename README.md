# Fake-News-Detection

# Fake News Detection using Machine Learning

## Overview

This project predicts whether a news article is REAL or FAKE using Natural Language Processing (NLP) and Machine Learning techniques.

## Features

* Fake news prediction
* Text preprocessing
* TF-IDF vectorization
* Logistic Regression model
* Real-time news classification

## Technologies Used

* Python
* Scikit-learn
* Pandas
* NLP
* Machine Learning

## Machine Learning Concepts

* Text Classification
* TF-IDF Vectorization
* Logistic Regression

## Accuracy

Achieved approximately 98% prediction accuracy on the dataset.

## Files

* `train_model.py` → Model training code
* `predict_news.py` → Prediction system
* `model.pkl` → Saved trained model
* `vectorizer.pkl` → Saved TF-IDF vectorizer

## How to Run

Install dependencies:

pip install pandas scikit-learn

Run the prediction program:

python predict_news.py


## Dataset

Fake and Real News Dataset from Kaggle.
